# 🛍️ E-com-Website

A modern and fully responsive **E-Commerce Website** built using **HTML, CSS, and JavaScript** with a premium UI/UX design.

This project includes beautiful product sections, dark/light mode, shopping cart functionality, smooth animations, customer reviews, and a clean responsive layout for all devices.

---

## ✨ Features

- 🖥️ Fully Responsive Design
- 🌙 Dark & Light Mode
- 🛒 Add to Cart Functionality
- 💳 Interactive Payment Methods
- ⭐ Customer Reviews & Ratings
- 🎨 Modern Premium UI Design
- ⚡ Smooth Animations & Effects
- 📱 Mobile Friendly Layout
- 🔍 Clean Navigation & Hero Section

---

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript
